# ImageDataExtractor
Extract images from **.accountpicture-ms** files

## Usage:
	`<executable name> <file name> <format ("PNG" or "JFIF")> <output file name (optional)>`
For example: `ImageDataExtractor.exe myFile.accountpicture-ms "PNG" extractedImage`
